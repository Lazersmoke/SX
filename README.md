# SX
SX is a Python based CodeGolfing language that includes many shortcuts to useful Python items

Here is a table showing the replacements:

    @@      self
    @       self.
    ++      +=1
    --      -=1
    ^       **
    j*      join
    p*      print
    e*      eval
    in*     input
    r*      replace
    $*      sys
    o&      open
    im*     import
    up&     upper
    low&    lower
    fl*     math.floor
    ce*     math.ceil
    &m      math
    &$      str
    i&      int
    g`      global
    c`      class
    m`      main
    d`      def
    e;      else
    ei;     elif
    %;      float
    l;      len
    M`      Main
